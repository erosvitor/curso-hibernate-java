## Sobre
Mapeamento objeto-relacional utilizando Hibernate e a linguagem Java.

## Tópicos
* Modelagem de dados
* Mapeamento simples
* Inserção
* Seleção
* Alteração
* Exclusão
* DAO genérico
* Mapeamento 1 para muitos
* Mapeamento muitos para muitos
* Entity Manager
